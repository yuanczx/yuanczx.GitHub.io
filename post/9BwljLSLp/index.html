Gradle构建Spring Boot+Kotlin

2021-11-22 Spring Boot ， Kotlin

使用Gradle构建Kotlin+Spring Boot项目遇到的一些问题

  1. 在@Application注解下使用open class而不是class

  2. 要使用data class作为entity需要在启动类加上注解
    @EnableConfigurationProperties([ClassName]::class)
    并且在entity上加注解：@ConstructorBinding

  3. 无法打包生成jar包。需要在build.gradle.kts文件中添加以下内容：

tasks.jar {
    manifest {
        attributes(mapOf("Main-Class" to "TestApplicationKt"))
    }
}
