Docker 常用命令及问题解决

2021-12-12 Docker

常用命令

Command Description
docker ps 列出正在运行的容器
docker ps -a 列出所有容器
docker run --name [name] -d -v [mPath]:[sPath] [image] 运行容器 -d后台 -v文件挂载 -p 端口映射
docker exec -it [name/id] /bin/bash 进入容器
docker start [name] 启动容器
docker build -t [tag] . 构建镜像 -t TAG

常见问题及解决方法

  • Docker 出现权限问题（Ubuntu）Got permission denied
    执行命令：sudo chmod a+rw /var/run/docker.sock

  • 在WSL2下无法通过IP地址访问Docker
    解决方法：配置端口转发 在PowerShell中执行如下命令（请勿换行）

    netsh interface portproxy add v4tov4     \
  listenport=22 listenaddress=0.0.0.0  \
  connectport=22 connectaddress=`WSL2 IP Address`

  • 在Ubuntu中升级Docker
    apt-get -u -y upgrade lxc-docker 安全升级Docker
    apt-get -u -y dist-upgrade lxc-docker 强制升级Docker
