Vue3 使用记录

2021-11-29 Vue ， Javascript

  • 添加点击事件：

    <button @click="click">Click me</button>

    export default{
    method:{
        click(){
            alert('You click the button')
        }
    }
}

  • 元素Attribute 绑定

    动态改变CSS

    <div :class="cls" @click="click">Test</div>

    export default{
    data(){
        return{
            cls:'ta'
        }
    },
    method:{
        click(){
            if(this.cls=='ta')
            this.cls='tb';
            else this.cls='ta';
        }
    }
}

    .ta{
    font-size:20px;
}
.tb{
    font-size:30px;
}

  • 计算属性使用方法

    <span>{{ publishedBooksMessage }}</span>  

    export default{
    data(){
        return{
            book:""
        }
    },
    computed:{
        publishedBookMessage:{
            get(){
                return this.book
            },
            set(newValue){
                this.book = newValue
            }
        }
    }
}

  • 侦听器

    <input v-model="test"/>

    export default{
    data(){
        return{
            test:""
        }
    }
    watch:{
        test(newValue, oldValue){
            alert(newValue);
        }
    }
}

  • 注意事项：

    • 使用Vite构建时需要使用在Compoent时需要加上.vue后缀
    • 在使用public文件夹下的资源时使用绝对路径
    export default{
    data(){
        return{
            src: '.public/[name].html'
        }
    }
}
