Python使用requests库请求网页并保存时出现编码错误

2021-05-13 Python

代码如下

import requests

if __name__ == '__main__':
    baseUrl = 'https://fanyi.baidu.com/?aldtype=85#en/zh/'
    text = input()
    response = requests.get(baseUrl + text)
    file = open(file="html.txt", mode="w+")
    file.write(response.text)
    file.close()

运行结果

原因：open 方法默认编码方式为 GBK 而我们请求的网页编码方式为 UTF-8

更改代码如下

file = open(file="html.txt", mode="w+", encoding="utf-8")

再次运行：正常
