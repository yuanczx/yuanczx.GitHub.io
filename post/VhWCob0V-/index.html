Linux 文件权限

2021-11-21 Linux

在 Linux 下使用ls -l命令查看文件类型以及权限

符号 类型
d 目录（文件夹）
l 链接（快捷方式）
c 字符设备，是一些串行端口的接口设备，例如键盘、鼠标、打印机、tty终端
b 块设备，是一些提供系统存取数据的接口设备，例如硬盘
s 套接字文件（socket），用于进程之间的通信
- 文件，分纯文本文件（ASCII）和二进制文件（binary）
符号 权限 数字
r 读 100(4)
w 写 010(2)
x 执行 001(1)
who 用户类型 说明
u user 文件所有者
g group 文件所有者及其所在组
o others 所有其他用户
a all 所有用户

示例 chmod a+rw /var/run/docker.sock为所有用户添加读写权限。

使用chmod [数字] file更改文件权限如：chmod 700 test.txt其中7为4+2+1=7
7：所属用户权限 0：所属用户组权限 0：其他用户权限
