在ViewModel中获取Context

2021-10-13 Jetpack ， Kotlin ， Android
  • 问题：在ViewMdel中并不建议我们持有Activity等实例，因为二者生命周期并不相同可能造成内存泄漏。因此当我们在ViewModel中要使用context时应该实时获取。
  • 解决方法：使用AndroidViewModel 中 getApplication().applicationContext实时获取context
  • Code
class MyViewModel(application:Application):AndroidViewModel(application){
    fun context = getApplication<Application>().applicationContext.
}
//当需要使用context时调用context方法即可
