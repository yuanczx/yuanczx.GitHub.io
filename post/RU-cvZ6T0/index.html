Neovim 的简单配置

2022-05-05 美化 ， Linux ， Vim

提示

  • Neovim 目录：~\AppData\Local\nvim\以及~\AppData\Lcoal\nvim-data
  • Neovim 提倡我们使用lua编写脚本

安装Neovim

  • Neovim官网
  • 点击 InstallNow
  • 根据提示下载安装 Windows 可以选择使用命令 winget install Neovim.Neovim 进行安装

安装插件管理工具

这里选用插件管理工具packer.nvim

  • 在Windows使用以下命令

    git clone https://github.com/wbthomason/packer.nvim "$env:LOCALAPPDATA\nvim-data\site\pack\packer\start\packer.nvim"

  • 在~\AppData\Local\nvim\lua\新建plugins.lua内容如下：

    return require('packer').startup(function()
-- Packer can manage itself
use 'wbthomason/packer.nvim'
end)

  • 在~\AppData\Local\nvim\新建init.lua内容如下：

    require('plugins')

  • 常用命令：

  • :PackerInstall 安装插件

  • :PackerUpdate 更新插件

  • :PackerSync 同步插件

安装插件（代码补全插件coc.nvim）

注意本步骤需要使用 yarn

  • GitHub 地址coc.nvim
  • 编辑plugins.lua
    加入use 'neoclide/coc.nvim'
    执行命令:PackerInstall
  • 进入目录~\AppData\Local\nvim-data\site\pack\packer\start\coc.nvim
    执行命令yarn install&&yarn build
  • 安装 coc-pyright(Python代码提示扩展)
    • CocInstall coc-pyright
  • 至此安装完成
