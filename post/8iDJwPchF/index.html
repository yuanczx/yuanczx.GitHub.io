Ubuntu Gnome桌面美化

2021-12-19 Ubuntu ， 美化 ， Linux

1. 安装 Tweaks (Gnome美化插件)

sudo apt update&&sudo apt install gnome-tweaks chrome-gnome-shell

2. 安装Tweaks插件

  • 下载地址
  • 安装 UserThemes

3. 选择喜欢的主题并安装

  • GnomeLook
  • 选择喜欢的主题下载
  • 安装主题
    将文件解压并将文件夹移动至对应目录如下表
    类型 目录
    主题 /usr/share/themes
    图标 /usr/share/icons
    光标 /usr/share/icons

4. 使用主题

  • 进入 Tweaks 选择外观Tab
  • 安装后需要切换一次主题才能显示新安装的主题
  • 最终效果图

5. 个人使用插件

插件 描述
Applications Menu 在顶栏显示APP列表
GTK Title Bar 隐藏标题栏
Hide Top Bar 隐藏顶栏
ibus font setting 输入法设置
User Themes 主题
