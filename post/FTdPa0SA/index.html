Nginx的简单使用

2021-11-25 Nginx ， Linux

配置

  1. 启动Nginx：nginx
  2. 重新加载配置文件nginx -s reload
  3. 负载均衡简单配置：
    http{
     upstream [name]{
         server ip weight=1;
         server ip weight=2;
     }

     server{
      location / {
             proxy_pass http://name;
             # root /usr/share/nginx/html;
             # index index.html index.htm;
     }
     }
}
  4. 使用VueRouter配置：
     location / {
         root /usr/share/nginx/html;
         index index.html index.htm;
         try_files $uri $uri/ @router;
     }
 location @router{
         rewrite ^.*$ /index.html last;
 }
